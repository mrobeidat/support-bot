# Return & Refund Policy

## Return Window
- You can return any item within 30 days of delivery
- Items must be in original condition with tags attached
- Original packaging is required for electronics

## How to Start a Return
1. Contact our support team with your order number
2. We will provide a prepaid return shipping label
3. Drop off the package at any authorized shipping location
4. Refund is processed within 5-7 business days after we receive the item

## Refund Methods
- Refunds are issued to the original payment method
- Store credit is available as an alternative (with a 10% bonus)
- Exchanges are free of charge

## Non-Returnable Items
- Gift cards
- Personalized/custom items
- Items marked as "final sale"
- Opened software or digital products

## Damaged or Defective Items
- Report damaged items within 48 hours of delivery
- We will send a replacement at no cost
- No need to return the damaged item for orders under $50
